你好，欢迎来到 eatmoreduck’s Blog。

这是一个个人博客，口号是 Code & Lifestyle & AI – 写代码、过日子、折腾 AI，大致就是这里会聊到的东西。

关于我#

我是 eatmoreduck，一个喜欢折腾技术和工具的开发者。平时主要写代码，偶尔搞搞 NAS 和智能家居，最近沉迷于用 AI 提升效率。

这个博客聊什么#

博客的内容主要围绕几个方向：

  • 编程开发：Java 后端、CI/CD、Git 工作流等实践经验分享
  • AI 工具：大模型应用、AI 辅助编程、提效工具的实际使用体验
  • NAS 与智能家居：群晖折腾记、远程访问方案、家庭网络搭建
  • 生活随笔：偶尔记录一些想法和碎碎念

写博客的初衷很简单 – 把自己踩过的坑和学到的东西记录下来，如果能顺便帮到别人就更好了。

技术栈#

这个博客用到的工具和框架：

  • 博客框架：Hugo – 静态站点生成器，构建速度快
  • 主题：PaperMod – 简洁的阅读体验
  • 编辑器：Obsidian – 用 Markdown 写文章，本地管理很方便
  • 部署：Cloudflare Pages + GitHub Pages 双平台
  • 评论系统：Giscus – 基于 GitHub Discussions
  • 图床：GitHub + jsDelivr CDN
