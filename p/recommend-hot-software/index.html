推荐一波MacOS 常用开发类软件，也为自己做一个备份，方便日后查找

开发工具类#

  1. jetbrain 全家桶 [[https://www.jetbrains.com/]] 非开源
  2. vscode https://code.visualstudio.com/
  3. curosr https://cursor.sh/
  4. Another Redis Desktop Manager https://github.com/qishibo/AnotherRedisDesktopManager
  5. Navicat https://www.navicat.com.cn/products#navicat-premium-lite 官方免费版本，无需破解
  6. sublime text https://www.sublimetext.com/
  7. Docker https://www.docker.com/
  8. DevToys https://devtoys.app/
  9. EasyDevo https://easydevo.boringboring.design/

实用工具类#

  1. 安卓文件传输 https://www.android.com/filetransfer/
  2. Neat Download Manager https://www.neatdownloadmanager.com/index.php/en/ 下载器
  3. Mounty https://mounty.app/
  4. keka https://www.keka.io/en/ 解压缩软件
  5. easydict https://github.com/tisfeng/Easydict 词典软件
  6. iina https://iina.io/ 全能视频播放器
  7. karabiner https://karabiner-elements.pqrs.org/ 按键修改
  8. monitor control https://github.com/MonitorControl/MonitorControl 外接显示器亮度调节
  9. hidden bar https://github.com/dwarvesf/hidden 菜单折叠
  10. ToothFairy https://apps.apple.com/us/app/toothfairy/id1191449274?mt=12 窗口管理
  11. Mac Mouse Fix https://macmousefix.com/ 鼠标平滑滚动
  12. PicList https://github.com/Kuingsmile/PicList 图床工具
  13. Loop https://github.com/MrKai77/Loop 窗口快捷放大缩小
  14. mos https://github.com/Caldis/Mos 鼠标平滑滚动利器
  15. ShortcutDetective https://formulae.brew.sh/cask/shortcutdetective 快捷键冲突排查软件
  16. Macs Fan Control https://crystalidea.com/macs-fan-control 风扇转速调节

效率类#

  1. alfred https://www.alfredapp.com/ 非开源
  2. 滴答清单 https://dida365.com/
  3. RayCast https://www.raycast.com/ 免费神作，可以作为 alfred 的替代

终端#

  1. iTerm2 https://iterm2.com/ 终端
  2. SecureCRT https://www.vandyke.com/products/securecrt/ 非开源
  3. lux https://github.com/iawia002/lux 免费下载器
  4. Termius https://termius.com/s 免费终端
  5. htop https://htop.dev/ 升级版 top
