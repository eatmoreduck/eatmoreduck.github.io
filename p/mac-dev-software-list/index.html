Mac 开发常用软件清单

整理一下自己 Mac 上常用的开发软件，算是给自己做个备份，顺便分享给有需要的人。
April 7, 2024  · eatmoreduck

换了几次电脑，每次都要重新装软件，干脆整理一下。以下是我在日常开发中真的在用的工具，不搞虚的。

开发工具#

软件说明
JetBrains 全家桶GoLand 用得最多，其他看项目需要
VSCode轻量编辑，前端开发主力
Cursor最近在用，AI 辅助写代码确实省脑子
Another Redis Desktop ManagerRedis 客户端，免费够用
Navicat Premium Lite官方免费版，不需要破解了
Sublime Text快速打开文件看代码
Docker容器开发必备
DevToys开发者小工具集合，JSON 格式化、Base64 转换啥的
EasyDevo和 DevToys 类似，二选一就行

实用工具#

软件说明
Android File Transfer传文件到手机，官方工具
Neat Download Manager下载器，比浏览器自带的快
Mounty读写 NTFS 格式硬盘
Keka解压缩，支持格式多
Easydict划词翻译，开源免费
IINA视频播放器，Mac 上最好的选择
Karabiner-Elements改键位，比如把 Caps Lock 映射成 Ctrl
MonitorControl外接显示器调节亮度，这个很有用
Hidden Bar折叠菜单栏图标，强迫症福音
Mac Mouse Fix鼠标滚动变平滑，用罗技鼠标的推荐
Loop窗口快捷缩放，类似 Windows 的 Aero Snap
Mos鼠标平滑滚动，二选一
PicList图床工具，支持 PicGo 配置
ShortcutDetective查看快捷键冲突，偶尔用到
Macs Fan Control风扇转速调节，笔记本发热时用

个人推荐：MonitorControl 和 Mac Mouse Fix/Mos 是真心推荐，用完就回不去了。

效率工具#

软件说明
Alfred老牌启动器，Powerpack 需要付费
滴答清单GTD 待办事项，多端同步
RayCast强烈推荐，免费版已经够用，Alfred 的平替

RayCast 真的好用，我主要是用来：

  • 快速启动应用
  • 查剪贴板历史
  • 简单的计算和转换
  • 搜索文件

终端工具#

软件说明
iTerm2终端替代品，配色和分屏都很方便
SecureCRT商用终端，付费
luxB 站视频下载，偶尔用
TermiusSSH 客户端，多设备同步
htoptop 的升级版，更好看

网络工具#

软件说明
Lulu开源防火墙，监控应用网络连接

Chrome 插件#

插件说明
Proxy SwitchyOmega代理切换，老牌了
ZeroOmegaSwitchyOmega 的轻量版
Adblock Plus广告拦截
简阅阅读模式，网页变清爽
Simple Allow Copy复制禁止复制的网站内容
XSwitch代理管理
Axure RP原型预览
沉浸式翻译页面双语对照翻译
Save All Resources下载网站资源，扒网站用

补充说明#

上面的工具不是都要装，按需选择就行。如果你也在整理自己的工具清单，推荐看看 awesome-mac，那里更全。
