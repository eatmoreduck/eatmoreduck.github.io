Mac 开发常用软件清单

换了几次电脑，每次都要重新装软件，干脆整理一下。以下是我在日常开发中真的在用的工具，不搞虚的。 开发工具 软件 说明 JetBrains 全家桶 GoLand 用得最多，其他看项目需要 VSCode 轻量编辑，前端开发主力 Cursor 最近在用，AI 辅助写代码确实省脑子 Another Redis Desktop Manager Redis 客户端，免费够用 Navicat Premium Lite 官方免费版，不需要破解了 Sublime Text 快速打开文件看代码 Docker 容器开发必备 DevToys 开发者小工具集合，JSON 格式化、Base64 转换啥的 EasyDevo 和 DevToys 类似，二选一就行 实用工具 软件 说明 Android File Transfer 传文件到手机，官方工具 Neat Download Manager 下载器，比浏览器自带的快 Mounty 读写 NTFS 格式硬盘 Keka 解压缩，支持格式多 Easydict 划词翻译，开源免费 IINA 视频播放器，Mac 上最好的选择 Karabiner-Elements 改键位，比如把 Caps Lock 映射成 Ctrl MonitorControl 外接显示器调节亮度，这个很有用 Hidden Bar 折叠菜单栏图标，强迫症福音 Mac Mouse Fix 鼠标滚动变平滑，用罗技鼠标的推荐 Loop 窗口快捷缩放，类似 Windows 的 Aero Snap Mos 鼠标平滑滚动，二选一 PicList 图床工具，支持 PicGo 配置 ShortcutDetective 查看快捷键冲突，偶尔用到 Macs Fan Control 风扇转速调节，笔记本发热时用 个人推荐：MonitorControl 和 Mac Mouse Fix/Mos 是真心推荐，用完就回不去了。 ...
